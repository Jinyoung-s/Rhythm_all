# 리듬 잉글리시 - 확장 커리큘럼 구조 (220개 패턴 대응)

## 설계 철학
- **노래 중심**: 각 스텝은 완성도 높은 한 곡의 노래
- **테마 기반**: 상황/감정/스토리 중심 구성
- **패턴 내포**: 문법은 자연스럽게 녹아들고, 학습 후 태깅
- **장르 다양성**: 테마에 맞는 최적의 음악 장르 선택
