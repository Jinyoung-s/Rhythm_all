## 📱 앱 개요

### 핵심 컨셉
**"음악 리듬으로 영어를 배우는 게임화된 학습 플랫폼"**

### 주요 기능
| 기능 | 설명 | 학습 효과 |
|------|------|----------|
| **리듬 게임 (Game1/Game2)** | 음악에 맞춰 노트 터치 | 리스닝, 발음 리듬 학습 |
| **SingAlong (따라 부르기)** | 실시간 음성 인식 | 스피킹, 발음 교정 |
| **StepTest (문장 조립)** | 단어 배열 테스트 | 문법, 어순 학습 |
| **VocabularyTest (단어 테스트)** | 단어 암기 퀴즈 | 어휘력 향상 |
| **Music Player** | Spotify 스타일 복습 플레이어 | 반복 학습, 리텐션 |
| **경제 시스템** | 포인트 획득 → 노래 구매 | 게임화, 동기부여 |

### 타겟 사용자
- **주 타겟**: 10-30대 (청소년 ~ 젊은 성인)
- **부 타겟**: 음악 학습을 선호하는 모든 연령층
- **페르소나**: 
  - "전통적인 영어 학습에 지루함을 느끼는 학습자"
  - "K-POP, 팝송으로 영어를 배우고 싶은 사람"
  - "게임을 좋아하는 학습자"

## 🌍 시장 분석

### 1. 시장 규모 (TAM/SAM/SOM)

#### TAM (Total Addressable Market) - 전체 시장
```
글로벌 에듀테크 시장 (2025)
├─ 전체 규모: $340B (약 450조원)
├─ 언어 학습 시장: $60B (약 80조원)
└─ 영어 학습 시장: $40B (약 53조원)
```

#### SAM (Serviceable Addressable Market) - 서비스 가능 시장
```
모바일 게임화 영어 학습 시장
├─ 전체 규모: $8B (약 10조원)
├─ 연평균 성장률 (CAGR): 18.7%
└─ 주요 지역: 한국, 일본, 중국, 동남아, 미국
```

#### SOM (Serviceable Obtainable Market) - 실제 목표 시장
```
리듬/음악 기반 언어 학습 (니치 시장)
├─ 추정 규모: $500M-$1B (6,500억-1.3조원)
├─ 성장 잠재력: ★★★★★ (매우 높음)
└─ 경쟁 강도: ★★★☆☆ (중간)
```
